
 Sentiment Analysis using Machine Learning

 Overview
This project predicts whether a text sentiment is Positive or Negative using Machine Learning.

 Features
- Text preprocessing
- TF-IDF Vectorization
- Model Training
- Sentiment Prediction
- Streamlit Web App

 Files
- app.py - Streamlit Application
- train.py - Model Training
- predict.py - Prediction Script
- model.pkl - Trained Model
- vectorizer.pkl - TF-IDF Vectorizer

 Technologies Used
- Python
- Scikit-learn
- Pandas
- NumPy
- Streamlit
